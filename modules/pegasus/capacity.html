What is Capacity?

The Capacity module evaluates your ability to perform two distinct tasks simultaneously: a high-precision Joystick Flying Task and a memory-based Visual Reference Task. It tests your cognitive workload limit and multi-tasking efficiency.

Joystick Task

GIF: Plane flying through the ball corridor

Visual Reference

Dice matching | Shape matching (8-pattern)

1. Joystick Task (Main Task)

  • Control a small plane using a joystick.
  • Fly inside a corridor made of orange and blue balls.
  • Keep the plane inside the corridor and avoid touching the sides.
  • Note: The corridor speeds up and slows down during the test.
  • You must maintain control while focusing on the second task.

2. Visual Reference Task

This task uses the joystick trigger and is divided into two halves:

  • First Half (Dice): A reference dice side is shown. If you see the same side again, pull the trigger. The reference changes once during this phase.
  • Second Half (Shapes): Rectangular shapes form a digital "8"-like pattern. If the two shapes shown on top of each other match, pull the trigger.

How to Prepare

  • Easy: Start here to learn both tasks and the joystick feel.
  • Medium: Similar to the hardest exam level.
  • Hard: Very challenging; builds strong confidence for the real test.
  • OFFICIAL Mode: Runs Phase 2 and Phase 3 under real exam conditions.
