Audio-Visual Memory

This test measures your ability to remember visual information and follow audio instructions at the same time.

Test Description

At the start of each question, you will see several corridors on the screen.

  • Two corridors are marked in red
  • Red corridors are occupied and cannot be used

Then, you will hear audio commands.

Each command includes:

  • A city name
  • A corridor number

If a corridor mentioned in the audio command was not red at the start, that corridor is valid.

This means the city connected to that corridor is a valid city.

If the corridor is red, it is occupied and cannot be used.

Your Task

  • Listen carefully to the audio commands
  • Remember which corridors were free
  • On the next screen, select the cities that use valid corridors
  • Only cities connected to free corridors should be selected.

Important Notes

  • There are always 2 occupied (red) corridors
  • In one question, there can be up to 6 audio commands
  • This means up to 6 valid cities
  • You will never need to select more than 6 cities

Phases

This test has 2 parts:

Practice Phase

  • First 5 questions
  • Not graded
  • Used to learn the rules

Exam Phase

  • All the following questions
  • Graded
  • No help or hints
